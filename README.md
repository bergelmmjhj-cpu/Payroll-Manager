# Running the Application

To start the backend and frontend, use the following commands:

1. Install the dependencies:
   ```bash
   pnpm install
   ```

2. Start the backend and frontend:
   ```bash
   pnpm dev
   ```

## Environment Variables
- `PORT`: The port number for the server to run on. Default is 3000.
- `BASE_PATH`: Base path for the application.

## Structure Overview
- **api-server**: Contains the backend API code.
- **payroll**: Contains the frontend application code.
- **scripts**: Utility scripts for development and deployment.
